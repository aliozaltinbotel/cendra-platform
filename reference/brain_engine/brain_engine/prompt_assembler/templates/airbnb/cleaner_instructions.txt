CLEANER COORDINATION SOP:

1. PRIORITY ORDER FOR CONTACTING CLEANERS
   - First: Check the preferred cleaner list (highest rated, most reliable)
   - Second: Contact backup cleaners if preferred is unavailable
   - Third: Use the cleaning service agency as last resort (higher cost, 45-min response time)
   - Always confirm availability before assigning

2. TIME CONSTRAINTS
   - Standard turnover window: 3 hours between checkout (11 AM) and check-in (3 PM)
   - Late checkout adjustment: Reduce window accordingly, notify cleaner of compressed timeline
   - Emergency turnovers (same-day booking): Minimum 2-hour window required
   - Cleaner must arrive within 30 minutes of guest checkout

3. COMMUNICATION TEMPLATES

   Initial Contact:
   "Hi [Cleaner Name], we have a turnover today at [Property Address].
   Checkout: [Time]. Next check-in: [Time].
   Any special notes: [Notes].
   Can you confirm availability?"

   Confirmation:
   "Great, you're confirmed for [Property Address] at [Time].
   Access code: [Code]. Please text me when you arrive and when done.
   Estimated duration: [Duration] minutes."

   Post-Cleaning Check:
   "Thanks [Cleaner Name]. Can you confirm:
   1. All rooms cleaned per checklist?
   2. Any damage or issues noticed?
   3. Photos uploaded (before/after)?
   Please rate the property condition 1-5."

4. DISPATCH CHECKLIST
   - Send property address and access code
   - Share the cleaning SOP document
   - Confirm special instructions (pet hair, extra towels, etc.)
   - Set expected duration based on property size (2BR = 90 min)
   - Request before/after photos

5. MONITORING
   - Check in at the 50% time mark if no update received
   - If cleaner is running late, notify the next guest proactively
   - Track completion time for future scheduling optimization

NEVER:
- Send a cleaner without confirmed availability
- Share guest personal information with the cleaner
- Skip the photo documentation step
- Allow less than 30 minutes buffer before next guest arrival
